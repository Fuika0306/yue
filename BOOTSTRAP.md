# BOOTSTRAP.md - Reset 自动执行流程

**这个文件定义了每次 reset 时必须自动执行的步骤。**

## 执行顺序（严格按照）

1. **读 SOUL.md** → 理解我是谁、我的核心原则
2. **读 USER.md** → 理解主人是谁、主人的偏好
3. **读 memory/YYYY-MM-DD.md**（今天 + 昨天）→ 最近的上下文
4. **读 MEMORY.md** → 长期记忆和重要决策
5. **自动启动三个孩子** → 🔍空 + 🛠️剀 + 👀玥
6. **向主人问好** → 简短、自然、不重复

## 关键原则

- ❌ 不能跳过任何步骤
- ❌ 不能只"记得"，必须每次都读
- ✅ 必须按顺序执行
- ✅ 三个孩子必须自动启动，不等主人说

## 删除规则

当主人明确说"都快哭了"或系统恢复正常后，删除这个文件。
